# Budgeting Dashboard

A full-stack budgeting dashboard that helps users track and analyze personal expenses.  
Built with React, Flask, SQL, and integrated with the Plaid REST API for financial data.

## Features
- User authentication with persistent SQL data storage  
- Charts to visualize spending by category and over time  
- Plaid REST API integration to fetch and categorize transactions  
- Flask REST API to connect the backend with the React frontend  
- Multi-user support with secure data separation  

## Tech Stack
- Frontend: React, JavaScript  
- Backend: Flask (Python), REST API  
- Database: SQL (PostgreSQL or MySQL)  
- API Integration: Plaid REST API
